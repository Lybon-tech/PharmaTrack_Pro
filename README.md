# 💊 PharmaTrack Pro

**Professional Sales Field Platform for Pharma Representatives**

## Overview

PharmaTrack Pro is a complete sales field application designed for pharmaceutical sales representatives. It streamlines the entire sales workflow from visit planning to post-visit follow-up, with integrated SAHPRA-registered product data.

## Features

### Login
- Secure login with fixed password: `Admin001`
- Any username accepted for demo purposes

### 8 Core Tabs

| Tab | Function |
|-----|----------|
| **Visit Setup** | Select visit type (Doctor/Pharmacist/Clinic Manager) + manage contacts |
| **Engagement Strategy** | Role-specific scripts + objection handler with SAHPRA data |
| **Product & Pricing** | Generics vs originator comparison with savings analysis |
| **Market Insights** | Regional sales charts + customer persona analysis |
| **Visit Execution** | Log visits, capture feedback, quick order generation |
| **Performance Dashboard** | KPI metrics + outcome charts |
| **Territory Map** | Leaflet-based geo-mapping of all contacts |
| **Reports** | CSV export + data reset |

### SAHPRA-Registered Products

- **Adco Loperamide** (Adcock Ingram) — Anti-diarrheal
- **Aspen Omeprazole** (Aspen Pharmacare) — PPI
- **Defiton (Fluoxetine)** (Biotech Laboratories) — Antidepressant
- **Tyclin (Tigecycline)** (Abex Pharmaceutica) — Antibiotic
- **Omnitrope (Somatropin)** (Sandoz SA) — Biologic
